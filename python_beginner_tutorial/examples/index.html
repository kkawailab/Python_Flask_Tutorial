Python サンプルコード集

11 章構成の「Python初級チュートリアル」で紹介しているサンプルコードをダウンロードできます。 ファイルを右クリックして保存するか、ブラウザ上でコードを表示してコピーしてください。

  • 01_hello_world.py

    最初の Python プログラム。標準出力とコメントの書き方を確認します。

  • 02_variables_and_types.py

    変数、基本データ型、型変換の基礎まとめ。

  • 03_strings.py

    文字列の操作方法、フォーマット、エスケープシーケンス。

  • 04_lists_and_tuples.py

    リスト・タプルの定義、メソッド、スライスの使い方。

  • 05_dictionaries_and_sets.py

    辞書と集合。キー操作や集合演算のサンプル付き。

  • 06_control_flow.py

    if, for, while 文など制御フロー構文を幅広く紹介。

  • 07_functions.py

    関数定義、デフォルト引数、可変長引数、スコープの解説。

  • 08_modules.py

    モジュールとパッケージの読み込み、標準ライブラリ活用例。

  • 09_classes_and_oop.py

    クラス定義、継承、特殊メソッドで学ぶオブジェクト指向。

  • 10_file_handling.py

    ファイルの読み書き、with 構文、例外処理と合わせた安全な操作。

  • 11_error_handling.py

    例外処理の基礎から独自例外までカバーするサンプル。
